## 1º Projeto: Contador.
# Contém:
- Botão + (Mais) p/ adicionar 1 (Um) ao número atual;
- Botão - (Menos) p/ remover 1 (Um) ao número atual;
- Botão Clear (Limpar) p/ remover qualquer número seja positivo ou negativo, deixando em seu estado de origem 0 (Zero).
